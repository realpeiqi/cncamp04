### 作业要求

1.接收客户端request，并将request 中带的header 写入response header
2.读取当前系统的环境变量中的VERSION 配置，并写入response header
3.Server 端记录访问日志包括客户端IP，HTTP 返回码，输出到server 端的标准输出
4.当访问localhost/healthz 时，应返回200
